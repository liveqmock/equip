## equip 机电设备管理系统


机电设备管理系统可以分为九大功能模块：采购管理，台账管理，库存管理，故障管理，维修管理，系统管理，权限管理，资料管理，报废管理。其中采购管理又可以分为：采购申请，申请查询，申请审批，下单，合同签订，设备验收，采购统计，申请取消，采购查询。台账管理可以分为：设备查询，查看履历。库存管理可以分为：出库，入库，出库统计，库存统计。
